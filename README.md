# 🌐 ReqRes API Testing Project

A comprehensive API testing collection for the **ReqRes.in** simulated API. This project demonstrates structured test organization and covers various HTTP methods (GET, POST, PUT, DELETE, PATCH).

## 🚀 Project Structure
The collection is organized into two main folders for better readability and execution:
* **Passed Requests:** Contains all GET methods for retrieving users and resources, with assertions for status codes and response data.
* **Functional Requests:** Covers User Management (Creation, Updates, Deletion) and Authentication (Login/Registration) scenarios.

## 🛠 Features
- **Dynamic Assertions:** Automated scripts to verify status codes and response bodies.
- **Full CRUD Coverage:** Testing Create, Read, Update, and Delete operations.
- **Negative Testing:** Scenarios for "User Not Found" and "Unsuccessful Login/Registration."

## ⚙️ How to Use
1. Download the `ReqRes.postman_collection.json` file.
2. Import it into **Postman**.
3. Run the collection using the **Collection Runner**.
